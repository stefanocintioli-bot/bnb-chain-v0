# BNB Chain v0 Brand Kit

Paste v0.md at the start of any v0 chat to get BNB Chain 
brand applied automatically. No need to re-explain colors, 
fonts or logos ever again.

WHAT'S IN HERE:
- v0.md — full brand system prompt, paste at start of every v0 session
- logos/ — all official BNB Chain SVG logos
